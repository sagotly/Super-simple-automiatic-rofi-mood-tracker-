# Super Simple Automatic Rofi Mood Tracker

Минималистичный трекер настроения, работающий через [rofi](https://github.com/davatorium/rofi) и cron.
Дважды в день (дневное и ночное окно) выскакивает диалог: сначала можно выбрать **заполнить сейчас** или **пропустить этот запуск**, затем — оценить состояние по шести шкалам от 1 до 10 и, при желании, добавить текстовые заметки (`notes`). Результаты сохраняются в CSV-файл для дальнейшего анализа.

## Зачем

Регулярная самооценка помогает замечать паттерны в настроении, тревожности и энергии. Данные копятся в простом CSV, который легко открыть в любой таблице, скормить скрипту визуализации или подгрузить в Obsidian/Grafana.

## Шкалы

| Метрика     | Что измеряет                         |
| ----------- | ------------------------------------ |
| **Mood**    | Общее настроение                     |
| **Apathy**  | Уровень апатии / безразличия         |
| **Health**  | Субъективное физическое самочувствие |
| **Swag**    | Уверенность в себе / «кайф от жизни» |
| **Anxiety** | Уровень тревожности                  |
| **Anger**   | Уровень раздражительности / злости   |
| **Notes**   | Свободный текст (опционально)        |

Все значения — целые числа от **1** (минимум) до **10** (максимум).

## Логика работы

### Временные окна

Сутки разбиты на два окна:

| Окно    | Начало  | Конец           |
| ------- | ------- | --------------- |
| Дневное | `12:00` | `19:00`         |
| Ночное  | `19:00` | `12:00` (след. день) |

В каждом окне допускается **ровно одна** запись. Cron запускает скрипт каждые 10 минут — если запись уже есть, скрипт тихо завершается.

### Порядок выполнения (`mood_prompt.sh`)

1. **Блокировка** — `flock` предотвращает одновременный запуск нескольких диалогов.
2. **Определение окна** — вычисляются границы текущего временного окна.
3. **Миграция CSV** — если файл содержит старый формат (5, 6, 7 или 8 колонок), он автоматически приводится к текущему (8 колонок, включая `notes`).
4. **Проверка дубликатов** — сканируется CSV на наличие записи в текущем окне.
5. **Стартовое меню** — в начале показывается выбор: `Start now` или `Skip this run`.
6. **Диалоги rofi** — последовательно запрашиваются шесть метрик (1–10), затем опциональные текстовые заметки.
7. **Повторная проверка** — защита от гонки: перед записью ещё раз проверяется, не появилась ли запись за время заполнения.
8. **Запись** — строка вида `dd-mm-yyyy HH:MM,mood,apathy,health,swag,anxiety,anger,"notes"` добавляется в CSV.

### Установка cron (`install_cron.sh`)

Скрипт добавляет в пользовательский crontab строку:

```
*/10 * * * * DISPLAY=:0 XDG_RUNTIME_DIR=... DBUS_SESSION_BUS_ADDRESS=... ~/.config/rofi/mood/mood_prompt.sh # rofi-mood-tracker
```

Переменные окружения (`DISPLAY`, `XDG_RUNTIME_DIR`, `DBUS_SESSION_BUS_ADDRESS`) нужны для того, чтобы rofi мог отобразить окно из контекста cron.

## Зависимости

| Пакет / утилита | Для чего                                       |
| --------------- | ---------------------------------------------- |
| **bash**        | Интерпретатор скриптов                         |
| **rofi**        | GUI-диалоги для старта, оценок и `notes`       |
| **cron**        | Периодический запуск (любая реализация: cronie, fcron, systemd-timer) |
| **flock**       | Блокировка файла (пакет `util-linux`)          |
| **GNU coreutils** | `date`, `mktemp`, `printf`                   |
| **sed / awk**   | Миграция старых CSV-форматов                   |

### Установка зависимостей (Arch / Arch-based)

```bash
sudo pacman -S rofi cronie util-linux
```

### Установка зависимостей (Debian / Ubuntu)

```bash
sudo apt install rofi cron util-linux
```

## Установка

```bash
git clone https://github.com/sagotly/Super-simple-automiatic-rofi-mood-tracker-.git ~/.config/rofi/mood
cd ~/.config/rofi/mood
chmod +x mood_prompt.sh install_cron.sh
./install_cron.sh
```

## Использование

### Запуск вручную

```bash
~/.config/rofi/mood/mood_prompt.sh
```

### Принудительный запуск (игнорирует лимит «одна запись на окно»)

```bash
~/.config/rofi/mood/mood_prompt.sh --force
```

### Удаление cron-задачи

```bash
crontab -l 2>/dev/null | grep -v 'rofi-mood-tracker' | crontab -
```

## Тема rofi

Скрипт использует файл темы `$HOME/.config/rofi/theme.rasi`. Если у вас другой путь или вы хотите использовать стандартную тему — отредактируйте переменную `THEME` в `mood_prompt.sh`.
